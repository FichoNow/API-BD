# API-BD

API desarrollada en Node.js + TypeScript.

## Requisitos

- npm >= 10
- Node: >= 20

## Instalación

Instalar dependencias:

`npm install`

## Scripts

### Desarrollo

`npm run dev`

Ejecuta el servidor en modo desarrollo.

### Build

`npm run build`

Compila el proyecto a la carpeta `dist/`.

### Producción

`npm run start`

Ejecuta la versión compilada desde `dist/`.

### Tests

`npm run test`

Ejecuta los tests de integración E2E. Requiere MySQL corriendo con la base de datos inicializada.
