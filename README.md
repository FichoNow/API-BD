# API-BD

API desarrollada en Node.js + TypeScript.

## Requisitos

- Node.js >= 20
- npm >= 10

Comprobar versiones:

## Instalación

Instalar dependencias:

`npm install`

## Scripts

### Desarrollo

`npm run dev`

Ejecuta el servidor en modo desarrollo.

### Build

`npm run build`

Compila el proyecto a la carpeta `dist/`.

### Producción

`npm run start`

Ejecuta la versión compilada desde `dist/`.
